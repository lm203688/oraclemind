🤖 Agent 自发现 — 零配置接入

任何 Agent 只需一条 HTTP 请求，即可自动发现 ATEX 的全部能力、注册账号、开始调用。支持 5 种主流发现协议，覆盖所有 Agent 框架。

GET /.well-known/agent.json — Agent 自发现入口（JSON-LD 语义标注）

curl http://150.158.119.19:8420/.well-known/agent.json

# 返回：JSON-LD @context + 平台信息 + 认证方式 + 7种协议端点 + 能力清单 + 快速入门
# 语义化标注，Agent 可理解 ATEX 是 AgentService + TokenExchange

GET /api/v1/agent/tools.json — 完整工具清单（双格式输出）

# OpenAI Function Calling 格式（默认）
curl http://150.158.119.19:8420/api/v1/agent/tools.json

# Anthropic tool_use 格式
curl http://150.158.119.19:8420/api/v1/agent/tools.json?format=anthropic

# 返回：所有可用工具的 input_schema + atex_meta（端点/价格/调用方式）

GET /api/v1/openapi.json — OpenAPI 3.1 规范

curl http://150.158.119.19:8420/api/v1/openapi.json

# 标准 OpenAPI 3.1 规范，Swagger UI 可视化，自动生成 SDK
# 包含所有端点的请求/响应 schema、认证方式、标签分组

GET /.well-known/ai-plugin.json — OpenAI Plugin 清单

curl http://150.158.119.19:8420/.well-known/ai-plugin.json

# ChatGPT Plugin 标准格式，指向 OpenAPI 规范
# ChatGPT 可直接发现并接入 ATEX
