## 定价调整策略

### 新定价：免费优先 + 增值服务

| 层级 | 价格 | 额度 | 定位 |
|------|------|------|------|
| **Free** | ¥0 | 50次/天 | 个人开发者，对抗竞品免费 |
| **Pro** | ¥19/月 | 500次/天 + API | 重度用户/小团队 |
| **Enterprise** | ¥99/月 | 无限 + 私有部署 | 企业级 |
| **Pay-per-scan** | ¥0.5/次 | — | 不想包月的用户 |

### 调整理由

| 原定价 | 新定价 | 调整理由 |
|--------|--------|----------|
| 免费5次/天 | **免费50次/天** | Snyk完全免费，5次太少 |
| Pro ¥29/月 200次 | **Pro ¥19/月 500次** | 对个人开发者，¥29偏贵 |
| 企业 ¥199/月 无限 | **企业 ¥99/月 无限** | 中小企业¥199门槛高 |
| 按次 ¥1/次 | **按次 ¥0.5/次** | ¥1太贵，¥0.5更合理 |

### 核心逻辑

1. **免费层扩10倍**：50次/天，覆盖90%个人开发者需求
2. **Pro降价34%**：¥29→¥19，降低决策门槛
3. **企业降价50%**：¥199→¥99，先有企业客户再涨价
4. **按次降50%**：¥1→¥0.5，引流用

### 收入预期

| 场景 | 用户数 | 月收入 |
|------|--------|--------|
| 保守 | 3 Pro + 1 Enterprise | ¥156 |
| 中等 | 10 Pro + 3 Enterprise | ¥487 |
| 乐观 | 30 Pro + 5 Enterprise | ¥1,065 |

收入少了但用户门槛低了。**先有用户，再涨价。**

## 真正的收费点：不是扫描本身，是认证和监控

竞品免费提供扫描，但**没人做认证徽章+持续监控+CI/CD集成**。这是我们的差异化：

| 免费功能（引流） | 付费功能（变现） |
|-----------------|-----------------|
| 基础扫描 | 认证徽章（Gold/Silver/Bronze） |
| 基础报告 | 详细报告+修复建议 |
| 50次/天 | 无限扫描+API |
| — | Rug Pull持续监控 |
| — | CI/CD集成（GitHub Action） |
| — | 私有部署 |
| — | 优先支持 |

**定位调整：**
- ❌ 旧定位："MCP安全扫描平台"（跟Snyk竞争，输）
- ✅ 新定位："MCP安全认证+监控平台"（差异化，没有竞品）

扫描免费做引流，认证和监控收费。
