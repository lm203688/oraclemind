📖 API文档

AIShield提供RESTful API，支持AI工具安全审计的完整流程。

快速开始

POST/api/v1/audit

提交审计请求。异步执行，立即返回audit_id，前端轮询获取结果。

请求体

{
  "tool_type": "mcp",          // 必填: mcp / skill / gpt / prompt
  "source_url": "https://github.com/owner/repo",  // 必填: GitHub URL
  "name": "My Tool",           // 可选: 工具名称
  "description": "..."         // 可选: 工具描述
}

响应

{
  "success": true,
  "audit_id": "audit_abc123",
  "status": "pending"          // pending → completed / failed
}

查询审计结果

GET/api/v1/audit/{audit_id}

查询审计结果。status为completed时返回完整报告。

响应（completed）

{
  "audit_id": "audit_abc123",
  "status": "completed",
  "name": "My Tool",
  "security_score": 96,        // 0-100
  "privacy_score": 100,        // 0-100
  "quality_score": 80,         // 0-100
  "performance_score": 100,    // 0-100
  "overall_score": 96,         // 加权总分
  "risk_level": "safe",        // safe / medium / high / critical
  "badge_level": "gold",       // gold / silver / bronze / none
  "findings": [...],           // 发现的问题列表
  "recommendations": [...],    // 安全建议
  "static_analysis": {...},    // 静态分析详情
  "dependency_analysis": {...},// 依赖分析详情
  "secrets_detection": {...},  // 敏感信息检测详情
  "semantic_analysis": {...},  // 语义分析详情（仅skill/gpt/prompt）
  "timestamp": "2026-06-09T12:00:00Z"
}

工具列表

GET/api/v1/tools

获取已审计工具列表，支持筛选和排序。

查询参数

qstring搜索关键词
tool_typestringmcp / skill / gpt / prompt
risk_levelstringsafe / medium / high / critical
badgestringgold / silver / bronze / none
sortstring排序字段（默认overall_score）
limitint每页数量（默认50，最大200）
offsetint偏移量

统计数据

GET/api/v1/stats

获取平台统计数据。

{
  "total_tools": 14,
  "total_audits": 40,
  "by_type": {"mcp": 13, "skill": 1},
  "by_risk": {"safe": 13, "critical": 1},
  "by_badge": {"gold": 13, "none": 1},
  "avg_security_score": 95.6

安全徽章

GET/api/v1/badge/{tool_key}

获取SVG格式安全徽章，可嵌入README。

安全40% 危险API、命令执行、网络请求、已知恶意依赖

质量20% 依赖数量、代码结构、文档完整性
性能15% 子进程使用、资源消耗、启动速度

集成方式

GitHub Action

# .github/workflows/aishield.yml
name: AIShield Audit
on: [push, pull_request]
jobs:
  audit:
    runs-on: ubuntu-latest
    steps:
      - uses: aisheild/audit-action@v1
        with:
          tool_type: mcp
          source_url: ${{ github.repositoryUrl }}

浏览器插件

安装AIShield浏览器插件，在GPT Store和MCP工具市场自动显示安全评分。

CLI

pip install aishield
aishield scan --type mcp --url https://github.com/owner/repo
