📡 基础信息

BASEhttp://8.217.147.255:8450/api/v1
所有API端点均支持CORS跨域访问，返回JSON格式数据。速率限制: 审计提交10次/分钟/IP

🔍 GET /api/v1/stats

获取平台统计数据：工具数、审计数、安全分分布等

📋 GET /api/v1/tools

查询已审计工具列表，支持筛选、搜索、排序
qstring搜索关键词（名称/URL）
tool_typestringmcp / skill / gpt / prompt
risk_levelstringsafe / low / medium / high / critical
badgestringgold / silver / bronze / none
sortstringoverall_score / security_score
limitint每页数量 (默认50, 最大200)
offsetint偏移量

🚀 POST /api/v1/audit

提交工具安全审计请求，异步执行扫描。返回audit_id用于轮询结果
tool_typestring必填: mcp / skill / gpt / prompt
source_urlstring必填: 工具源码URL (GitHub等)
namestring可选: 工具名称
descriptionstring可选: 工具描述

📄 GET /api/v1/audit/{audit_id}

获取审计报告详情。status=pending时表示扫描进行中，前端应3秒轮询

🕐 GET /api/v1/recent

💚 GET /api/v1/health

服务健康检查端点

🔄 扫描流程

AIShield采用6步流水线扫描，在独立subprocess中执行：
1. 源码拉取-GitHub API + raw URL双重获取，最多15个文件
2. 静态分析-22+检测模式：危险API、命令执行、网络请求、数据库操作
3. 依赖分析-npm/pypi依赖扫描，含已知恶意包检测
4. 敏感信息-API密钥/私钥/数据库连接串检测
5. 语义分析-比特助手越狱检测+数据外传识别(Skill/GPT/Prompt)
6. 四维评分-安全/隐私/质量/性能 → 综合分+徽章等级

🏆 徽章等级

🥇 Gold≥85分安全可信，推荐使用
🥈 Silver70-84分基本安全，需关注部分问题
🥉 Bronze50-69分存在风险，谨慎使用
⚠️ None<50分严重风险，不建议使用
