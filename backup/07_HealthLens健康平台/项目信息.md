# HealthLens 健康全景平台

## 项目概述
跨生态健康数据聚合器——OCR医学报告解析+健康分析+FHIR导出

## 服务
- 线上: http://150.158.119.19:8432
- 服务名: healthlens

## ECS部署
- 路径: /home/ubuntu/healthlens/
- 核心文件:
  - app.py / app_v05.py - Flask入口
  - health_analyzer.py - 健康分析引擎
  - data_connectors.py - 数据连接器
  - ocr_engine.py - OCR医学报告解析
  - fhir_exporter.py - FHIR标准导出
  - loinc_mapper.py - LOINC医学编码映射
  - security.py - 安全模块
  - pii_sanitizer.py - PII个人信息脱敏
  - logging_config.py - 日志配置
  - blueprints/ - Flask蓝图模块

## 功能
1. OCR医学报告解析（MedicalReportParser）
2. 健康数据分析
3. FHIR标准导出
4. LOINC医学编码映射
5. PII个人信息脱敏

## 技术栈
- Flask + SQLite + loguru结构化日志
- OCR引擎
- FHIR医疗数据标准
- LOINC医学编码
