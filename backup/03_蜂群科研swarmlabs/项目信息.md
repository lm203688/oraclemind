- POST /api/v1/register - 发送验证码

- POST /api/v1/login_pwd - 密码登录
- POST /api/v1/set_password - 设置密码
- POST /api/v1/experiment/submit - 提交实验
- GET /api/v1/experiment/{id}/progress - 实验进度
- GET /api/v1/report/{id} - 实验报告
- GET /api/v1/experiments/history - 历史实验
- GET /api/v1/physics/rules - 物理规则列表(248条)
- POST /api/v1/physics/evaluate - 物理规则评估
- GET /api/v1/quantum/{smiles} - 量子化学计算
- POST /api/v1/sandbox/exec - Docker沙箱执行
- GET /api/v1/benchmark - 评测基准
- POST /api/v1/assistant - AI助手
- GET /api/v1/reagent/{name} - 试剂查询
- GET /api/v1/patent/{name} - 专利查询
