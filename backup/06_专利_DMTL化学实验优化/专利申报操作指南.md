## 二、申报流程（两种方式）

### 方式一：电子申请（推荐）

#### 步骤1：注册账号

1. 打开 **专利业务办理系统**：https://cponline.cnipa.gov.cn
2. 点击"注册"
3. 选择"个人用户"或"法人用户"
4. 填写真实姓名、身份证号、手机号
5. 完成实名认证（人脸识别）
6. 登录系统

#### 步骤2：新建申请

1. 登录后点击"专利申请" → "新申请"
2. 选择申请类型：**发明专利**
3. 填写申请信息：
   - 发明名称：`一种基于DMTL闭环和Arrhenius方程的化学实验优化方法及系统`
   - 申请人：你的姓名
   - 发明人：你的姓名
   - 代理机构：无（自行申请）

#### 步骤3：上传材料

按顺序上传以下文件（PDF格式）：

1. **请求书**：在系统中在线填写（不用上传PDF），内容参考"专利申请文件_完整版.pdf"第2页
2. **权利要求书**：上传"专利申请文件_完整版.pdf"或单独提取权利要求书部分
3. **说明书**：上传"专利申请文件_完整版.pdf"
4. **说明书摘要**：上传"专利申请文件_完整版.pdf"最后一页
5. **附图**：依次上传5张附图PDF

**注意**：每个文件大小不超过20MB

#### 步骤4：确认提交

1. 检查所有信息无误
2. 勾选"请求费用减缴"（如符合条件）
3. 点击"提交"
4. 系统生成申请号（格式：20261XXXXXXXXXX）
5. 记录申请号

#### 步骤5：缴纳费用

1. 提交后在系统中点击"费用管理" → "待缴费用"
2. 缴纳申请费¥900（减缴后¥135）
3. 支持支付宝/微信/银行转账
4. 缴费成功后申请正式受理

## 三、提交后流程

### 时间线

| 阶段 | 时间 | 说明 |
|------|------|------|
| 受理 | 提交后1-3天 | 收到受理通知书，获得申请号 |
| 初审 | 1-3个月 | 形式审查（格式/材料完整性） |
| 公布 | 申请日起18个月 | 自动公布（可申请提前公布） |
| 实审 | 申请日起3年内 | 需提交实质审查请求+缴费¥2,500 |
| 授权 | 实审通过后2-4个月 | 收到授权通知书 |
| 领证 | 授权后1-2个月 | 缴纳登记费+印花税，领取证书 |

### 实质审查请求

- **必须在申请日起3年内**提交实质审查请求
- 提交"实质审查请求书"
- 缴纳实质审查费¥2,500（减缴后¥375）
- 可以在提交申请时同时请求实质审查（加快流程）

### 加快审查（如有需要）

- **PPH（专利审查高速路）**：如有外国同族专利已授权，可申请PPH加快
- **优先审查**：符合国家重点发展领域的可申请优先审查（6个月出结果）
- **提前公布**：申请时勾选"请求提前公布"，1-3个月即公布

## 四、常见问题

### Q1：个人申请还是公司申请？
- **个人申请**：费用低（可减缴85%），但专利权属于个人
- **公司申请**：费用高（不可减缴），但专利权属于公司，便于商业化
- **建议**：如用于商业化，用公司申请；如用于个人项目/职称评定，用个人申请

### Q2：需要找代理机构吗？
- **自行申请**：费用低，但需自己处理所有流程和答复
- **代理机构**：费用¥3,000-8,000，专业代理处理流程和答复审查意见
- **建议**：如对专利流程不熟悉，建议找代理（可找便宜的¥3,000左右）

### Q3：实质审查是什么？
- 实质审查是审查员检查你的发明是否满足：
  1. 新颖性（之前没有人做过）
  2. 创造性（不是显而易见的改进）
  3. 实用性（能在产业中应用）
- 审查员会发"审查意见通知书"，你可能需要答复（1-2次）
- 答复期限：4个月内（可延期2个月）

### Q4：如何答复审查意见？
- 收到审查意见后，仔细阅读审查员的反对理由
- 针对每条意见进行答复：
  - 同意的：修改权利要求
  - 不同意的：提供证据和理由反驳
- 如需修改权利要求，提交"修改替换页"
- **建议**：第一次审查意见答复建议找专利代理师帮忙

### Q5：专利保护期多久？
- **发明专利**：20年（从申请日起算）
- 每年需缴纳年费（第1-3年¥900/年，后续递增）

### Q6：可以申请国际专利吗？
- 可以，通过PCT国际申请
- 优先权期限：在中国申请后12个月内
- PCT申请费用约¥10,000+翻译费

## 本专利核心保护点

1. **3轮DMTL闭环**：广撒网→贝叶斯优化→精细搜索
2. **Arrhenius方程集成**：k=exp(-Ea/RT)计算速率常数
3. **催化剂效应矩阵**：5种催化剂差异化建模
4. **溶剂极性匹配**：5种溶剂差异化建模
5. **综合成功率公式**：热力学×0.3 + 速率×0.25 + 催化剂 + 溶剂 + 0.1
6. **独立Reviewer审查**：偏差超20%自动标记
7. **副产物预测**：高温分解/水解/原料残留
8. **逆合成规划**：目标分子分解为可购买原料
