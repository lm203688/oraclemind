# 小乌数据采集任务包 — 第一批

## 定时采集机制

**执行频率**：每小时1次，每次完成1个任务（30-50个实体）
**轮转方式**：按下方"执行顺序"列表依次推进，完成一个标记✅，继续下一个
**中断恢复**：每次启动时先检查进度（看哪些任务已产出JSON文件），从未完成的第一个继续
**产出交付**：每完成一批，将JSON文件保存到本地目录 `C:\Users\xing\小乌助手\定时任务收集结果\`，文件命名格式：`{任务编号}_{站点名}_{类别}_{日期时间}.json`，例如 `task08_exo-science_exoplanets_0629-1400.json`

### 定时任务prompt模板（设置给小乌的定时触发指令）

```

2. 对照任务清单（共14个任务），找到第一个未完成的任务
3. 执行该任务：
```
