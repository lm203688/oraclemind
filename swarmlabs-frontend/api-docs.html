POST/api/v2/optimize/{engine} — AI参数优化
GET/api/v2/validate/{engine} — 验证引擎精度
GET/api/v2/info/{engine} — 引擎信息
POST/api/v1/register — 注册免费API Key
GET/api/v1/health — 健康检查
