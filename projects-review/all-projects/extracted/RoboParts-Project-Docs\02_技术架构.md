## 四、API端点清单

### 端点1：兼容性查询（零成本）
```
GET/POST  /api/v1/compat-check
```
**功能**：查询指定零件与机械臂的兼容性
**实现方式**：查询Supabase兼容性数据库
**成本**：零成本（纯数据库查询）
**状态**：✅ 已实现

**请求示例**：
```json
{
  "arm_model": "UR5e",
  "gripper_model": "Robotiq 2F-85"
}
```

**响应示例**：
```json
{
  "compatible": true,
  "confidence": 0.95,
  "adapter_required": false,
  "notes": "原生兼容，无需适配器"
}
```

---

### 端点2：零件推荐（零成本）
```
GET/POST  /api/v1/parts-recommend
```
**功能**：根据机械臂型号推荐兼容零件列表
**实现方式**：查询Supabase + 规则排序
**成本**：零成本
**状态**：✅ 已实现

**请求示例**：
```json
{
  "arm_model": "UR5e",
  "part_type": "gripper",
  "budget_usd": 500
}
```

---

### 端点3：自然语言兼容性查询（LLM驱动）
```
POST  /api/v1/nl-compat
```
**功能**：接受自然语言问题，返回兼容性答案
**实现方式**：Agnes AI（agnes-2.0-flash）+ Supabase
**成本**：按Agnes AI用量付费
**状态**：🔄 环境变量已配置，需Redeploy后测试

## 五、Supabase数据库结构（推断）

### 主要数据表

**robot_arms（机械臂）**
```sql
id           UUID PRIMARY KEY
brand        TEXT          -- 品牌：UR, FANUC, KUKA...
model        TEXT          -- 型号：UR5e, M-10iA...
payload_kg   FLOAT         -- 载重(kg)
reach_mm     INT           -- 臂展(mm)
flange_type  TEXT          -- 法兰类型：ISO-9283, D50...
created_at   TIMESTAMP
```

**grippers（夹爪）**
```sql
id               UUID PRIMARY KEY
brand            TEXT          -- Robotiq, OnRobot...
model            TEXT          -- 2F-85, RG2...
type             TEXT          -- 平行夹爪/三指/吸盘
max_payload_kg   FLOAT
mounting_type    TEXT          -- 法兰规格
price_usd        FLOAT
affiliate_url    TEXT          -- 佣金链接
```

**compatibility（兼容性关系）**
```sql
id               UUID PRIMARY KEY
arm_id           UUID REFERENCES robot_arms
gripper_id       UUID REFERENCES grippers
compatible       BOOLEAN
adapter_required BOOLEAN
confidence       FLOAT         -- 0-1置信度
notes            TEXT
verified_by      TEXT          -- 社区/官方/测试
```

**stl_models（STL适配器文件）**
```sql
id           UUID PRIMARY KEY
arm_id       UUID REFERENCES robot_arms
gripper_id   UUID REFERENCES grippers
file_url     TEXT
price_cny    FLOAT
downloads    INT
```
