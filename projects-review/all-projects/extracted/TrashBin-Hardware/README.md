# 🗑️ 电动按钮压缩垃圾桶 — 硬件原型

> 设计阶段：概念验证 | 目标成本：300-500 RMB | 版本：v0.1

## 一、产品定义

一款**电动按钮式压缩垃圾桶**：按下按钮 → 电机驱动压板下压 → 压缩垃圾体积 → 自动回弹。

| 特性 | 描述 |
|------|------|
| 触发方式 | 按钮（物理按键，非传感器） |
| 压缩方式 | 电机+丝杆/齿条驱动压板 |
| 安全机制 | 电流检测防夹手、限位开关 |
| 供电 | 18650锂电池 ×4（或USB-C 5V） |
| 容量 | 12-15L标准垃圾袋 |

## 四、控制逻辑

```
状态机：IDLE → COMPRESSING → HOLD → RETURNING → IDLE

1. IDLE：等待按钮
2. [按下按钮] → COMPRESSING：电机正转，压板下压
3. [电流>阈值 OR 限位开关触发] → HOLD：停止，保持3秒
4. HOLD → RETURNING：电机反转，压板回升
5. [上限位开关触发] → IDLE
```
