# RoboLink - 机器人零件对接平台

> 不同品牌的机器人零件，在这里找到连接方式

## 这是什么？

RoboLink 是面向机器人创客、爱好者和中小集成商的**零件对接平台**，解决"不同品牌的机械臂和夹爪能不能接、怎么接"这个核心问题。

## 核心功能

### 1. 零件选型引擎
收录 15+ 品牌、34+ 型号的机器人机械臂和末端执行器参数，支持品类筛选（电动夹爪/柔性夹爪/舵机夹爪/真空吸盘）、品牌筛选和关键词搜索。

### 2. 兼容性检查器
选择机械臂型号 + 夹爪型号 → 自动输出：
- 法兰接口是否匹配
- 通讯协议是否兼容
- 是否需要转接件
- 推荐的STL转接件设计

### 3. STL转接件库
提供 12 款免费下载的3D打印转接件设计，覆盖常见跨品牌对接场景：
- ISO50法兰转M4转接板
- ISO50法兰转舵机支架
- 双夹爪安装板
- 线缆拖链安装支架
- 被动式快换接头
- uArm通用夹爪安装座
- ...更多持续更新

### 4. 3D模型在线预览
支持在网页上3D旋转查看转接件模型，无需下载。

### 5. 社区交流
创客们分享搭配方案、踩坑记录、改装教程。支持发帖、评论、点赞。

### 6. 3D打印代打
没有打印机？一键跳转嘉立创/魔猴网下单代打，15-50元/个，3-5天送达。

### 7. 行业动态监控
每日自动追踪国内外夹爪品牌新品、开源社区转接件设计、行业标准化动态。

## 技术栈

- 纯前端：HTML + CSS + Vanilla JavaScript
- 3D预览：Three.js
- 数据存储：localStorage（社区帖子）
- 自动化：定时任务自动更新监控数据
- 部署：静态站点
