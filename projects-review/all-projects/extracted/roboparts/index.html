🛠️ 开发者 API

RoboParts 开放零件兼容性和选型推荐 API，供 LLM、控制层软件、第三方工具调用。 所有端点免费，每天 1000 次配额。

🔑 获取 API Key

在 api@roboparts.cc 发送邮件申请，注明用途。格式：rbp_xxxxxxxx

📡 端点 1：兼容性检查

POST https://roboparts.cc/api/v1/compat/check
Headers:
  X-API-Key: rbp_xxxxxxxx
  Content-Type: application/json

Body:
{
  "robot_arm": "dobot-magician",
  "end_effector": "robotiq-2f85"
}

🤖 端点 2：零件推荐

POST https://roboparts.cc/api/v1/parts/recommend
Headers:
  X-API-Key: rbp_xxxxxxxx
  Content-Type: application/json

Body:
{
  "robot_arm": "dobot-magician",
  "task": "抓取易碎品",
  "budget": 2000,
  "prefer_brand": null
}
