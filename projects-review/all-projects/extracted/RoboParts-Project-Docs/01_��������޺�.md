# RoboParts 平台项目概览

> 文档版本：v1.0 | 更新日期：2026-06-25 | 负责人：厉兴

---

## 一、项目定位

**RoboParts（roboparts.cc）** 是一个机器人零部件兼容性平台，核心价值是解决机器人行业"零件选型难、兼容性不透明"的痛点。

| 维度 | 描述 |
|------|------|
| 域名 | roboparts.cc |
| 定位 | 机器人零部件兼容性查询 + 导购 + 社区平台 |
| 目标用户 | 机器人工程师、集成商、研究人员、爱好者 |
| 商业阶段 | 早期，尚未盈利，正在验证收入路径 |
| GitHub | lm203688/roboparts |

---

## 二、功能模块

| 模块 | 状态 | 描述 |
|------|------|------|
| 零件选型 | 规划中 | 按机械臂型号筛选兼容零件 |
| 兼容性校验 | ✅ API已实现 | 查询零件A与机械臂B是否兼容 |
| 用户社区 | 规划中 | 用户分享兼容方案、经验 |
| STL适配器库 | 规划中 | 3D打印适配器STL文件下载 |
| 3D打印服务 | 规划中 | 按需打印适配器零件 |
| 行业动态监测 | 规划中 | 机器人行业新品/新闻推送 |
| 联盟佣金体系 | 规划中 | 零件导购佣金分成 |
| 支付通道 | 🔄 调试中 | 虎皮椒支付（已接入，待调试） |
