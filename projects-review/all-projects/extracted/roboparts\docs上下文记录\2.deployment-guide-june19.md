## 六、问题排查

### Vercel 部署失败

- 检查 Vercel 构建日志：**Deployments** → 点击部署 → **Build Logs**
- 常见错误：`api/_data.js` 语法错误、Node.js 版本不匹配
- 解决方法：本地运行 `node api/_data.js` 检查语法

### Supabase SQL 执行失败

- 检查 SQL 语法（尤其注意 PL/pgSQL 函数的 `$$` 分隔符）
- 如果 `user_scores` 表创建失败，手动创建（见第三节）

### 前端页面功能不工作

- 打开浏览器开发者工具（F12），检查 Console 错误
- 常见错误：`js/community.js` 中 `RoboLinkAuth` 未定义（确保 `js/auth.js` 先加载）
- 解决方法：检查 `index.html` 的脚本加载顺序（config → data → auth → app → community）
