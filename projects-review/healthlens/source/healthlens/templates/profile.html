🔗 Webhook 接入

通过Webhook推送健康数据到HealthLens，支持任何设备或应用

POST /api/webhook/{your_user_id}
Header: X-API-Key: hl_live_xxxxx
Content-Type: application/json

{
  "source": "my_device",
  "format": "healthlens",
  "records": [
    {
      "metric_type": "心血管",
      "metric_name": "心率",
      "value": 72,
      "unit": "bpm",
      "recorded_at": "2026-06-09"
    }
  ]
}

也支持自定义格式：直接传 key-value 对，如 {"heart_rate": 72, "steps": 8000}
