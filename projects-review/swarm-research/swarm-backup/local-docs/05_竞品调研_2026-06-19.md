## 二、Top5直接竞品深度分析

### 1. AI-Scientist v1/v2 (SakanaAI) — 13,992 + 6,607 Stars

**能力：**
- v1: idea生成→代码修改→实验运行→论文撰写 全自动
- v2: 智能体树搜索(BFTS)，首篇AI论文通过同行评审(workshop级)
- 仅限ML/Diffusion领域，模板化实验

**架构：** 单Agent流水线，非多Agent协作
**部署：** 纯开源，自部署，需自备API Key
**商业化：** 无，Sakana靠融资不是SaaS

**vs蜂群：**
- ✅ 蜂群多Agent协作 > AI-Scientist单流水线
- ✅ 蜂群化学领域 > AI-Scientist仅ML
- ❌ AI-Scientist有Nature级学术背书，蜂群没有
- ❌ AI-Scientist已验证端到端能力，蜂群MVP未验证

**部署：** 开源+Live Demo(azurewebsites.net)，但非SaaS
**商业化：** 无直接商业化，微软内部使用

**vs蜂群：**

## 三、关键发现

### 发现1：全部开源，零SaaS

**20个竞品中，0个有SaaS/付费模式。** 全部是开源自部署。

这意味着：
- ✅ 蜂群的SaaS+积分制是**真正的蓝海**
- ⚠️ 用户习惯免费用开源工具，付费转化可能困难
- ✅ 开源工具部署门槛高(API Key/Docker/Python环境)，SaaS有天然优势

### 发现2：化学方向几乎空白

| 竞品 | 化学能力 |
|------|----------|
| AI-Scientist | ❌ 仅ML |
| RD-Agent | ❌ 量化/ML |
| AutoResearchClaw | ❌ CS/ML |
| AI-Researcher | ❌ 通用ML |
| InternAgent | ⚠️ 有反应产率/分子动力学，但非核心 |
| Biomni | ⚠️ 生物医学，偏生物 |

**化学虚拟实验室是蜂群最大的差异化机会。**

### 发现3：学术背书是护城河

| 竞品 | 学术背书 |
|------|----------|
| AI-Scientist v2 | 首篇AI论文通过同行评审 |
| AI-Researcher | NeurIPS 2025 Spotlight |
| InternAgent | 上海AI Lab, MLEBench #1 |
| AutoResearchClaw | arXiv论文 |
| Biomni | Stanford, bioRxiv |

蜂群目前**零学术背书**，这是最大劣势。

### 发现4：工程成熟度差距大

| 维度 | 竞品top5 | 蜂群 |
|------|----------|------|
| 测试覆盖 | AutoResearchClaw 2699测试 | 无测试 |
| Docker化 | 大部分有Dockerfile | 无 |
| Web UI | 多数有Gradio/Streamlit | 有(挂载AIShield) |
| 多模型支持 | LiteLLM/OpenRouter | 仅agnes |
| 持久记忆 | InternAgent有memory模块 | 有(简单JSON) |

### 发现5：蜂群化学蜂当前是"伪计算"

当前chemistry_bee.py的实际实现：
- 反应预测：LLMprompt生成，**没用RDKit/IBM RXN**
- 逆合成：LLMprompt生成，**没用AIZynthFinder**
- 性质预测：LLMprompt生成，**没用RDKit描述符**
- 量子化学：LLMprompt生成，**没用PySCF**

**架构文档写了用开源引擎，但代码实际全靠LLM猜。** 这是最大技术风险。

## 四、蜂群平台SWOT分析

### Strengths（优势）
1. SaaS即开即用，零部署门槛
2. 积分制商业模式清晰
3. 化学方向定位明确
4. 模块化叠加定价(免费引流+付费增值)
5. BYOK支持，降低用户成本

### Weaknesses（劣势）
1. 化学蜂实际未接入计算引擎（伪计算）
2. 零学术背书
3. 工程成熟度低(无测试/无Docker)
4. 仅支持agnes单一模型
5. MVP未验证端到端能力
6. 无社区运营(竞品都有Slack/Discord/飞书群)

### Opportunities（机会）
1. 竞品全开源零SaaS → SaaS蓝海
2. 化学虚拟实验室空白 → 先发优势
3. 国内市场(广东省AI赋能科研政策) → 政策红利
4. 积分制模糊定价 → 降低付费心理门槛
5. AIShield生态协同 → 安全审计差异化

### Threats（威胁）
1. InternAgent已覆盖化学/物理/生物 → 直接竞争
2. 开源工具免费习惯 → 付费转化难
3. 大厂入场风险(微软RD-Agent已13K stars) → 资源碾压
4. LLM能力趋同 → 技术壁垒低
5. 学术界认可度 → 没论文=没信任
