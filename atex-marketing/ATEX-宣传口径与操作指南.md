**获取API Key：**
1. 访问 https://genetech.tools/credits.html
2. 支付宝充值（最低1元起）
3. 获得API Key，余额永不过期

**调用示例：**
```bash

  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"service_id":"svc_046","args":{"text":"你的内容","platform":"douyin"}}'

# AI搜索可见度检测（2块钱）
```
