Qiskit Runtime (IBM)

quantum software
Qiskit Runtime is IBM's execution environment that runs quantum programs server-side on IBM Quantum hardware. It introduces the Primitives API (Sampler for sampling, Estimator for expectation values) that abstracts away hardware details and provides built-in error mitigation. Sessions enable efficient multi-circuit execution. Qiskit Runtime is central to IBM's quantum-centric supercomputing vision
