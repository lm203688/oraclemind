什么是基因治疗数据库？

基因治疗数据库（Gene Therapy Database）是系统收录基因治疗领域基因-疾病-疗法关系的结构化知识库。GeneTech Tools 通过自动采集PubMed论文、临床试验注册、行业新闻，经交叉验证后入库，确保数据准确性和时效性。
